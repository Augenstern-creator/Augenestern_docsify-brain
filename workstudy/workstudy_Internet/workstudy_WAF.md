# 1、Web应用防火墙WAF



## 1.1、什么是WAF

- Web应用防火墙（Web Application Firewall，简称WAF）
- 对网站或者App的业务流量进行恶意特征识别及防护，在对流量清洗和过滤后，将正常、安全的流量发送给服务器，避免网站服务器被恶意入侵导致性能异常等问题，从而保障网站的业务安全和数据安全。
