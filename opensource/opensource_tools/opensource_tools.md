# 1、Vercel

## 1.1、简介

1. Vercel 旨在为开发者提供简单直观的使用体验。通过简单的几步设置，用户可以快速将他们的项目部署到云端。
2. Vercel 支持从 GitHub、GitLab 或 Bitbucket 等代码仓库直接部署，当你推送代码到仓库时，Vercel 会自动进行构建和部署。
3. 官网：https://vercel.com/
